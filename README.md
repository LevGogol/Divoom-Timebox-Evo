# Divoom Timebox Evo

> **⚠ Work in progress.** Проект создан вайбкодом — код сырой, API нестабильное, а документация неполная.

Управление экраном Divoom Timebox Evo через Bluetooth: отрисовка изображений, голосовое управление цветом, голосовой таймер с визуальным обратным отсчётом.

## Требования

- Python 3.9+
- Windows 10/11 с Bluetooth-адаптером
- Divoom Timebox Evo

```powershell
pip install vosk pyaudio playsound==1.2.2
```

Модель распознавания речи (~45 МБ), скачать один раз:

```
https://alphacephei.com/vosk/models/vosk-model-small-ru-0.22.zip
```

Распаковать так, чтобы рядом со скриптами находилась папка `vosk-model-small-ru-0.22/`.

## Подключение колонки

### Шаг 1: Сопряжение

1. Включи Timebox Evo
2. Открой **Параметры Windows** → **Bluetooth и устройства** → **Добавить устройство**
3. Выбери **Bluetooth**
4. Найди **Timebox-Evo** в списке и нажми **Подключить**
5. Дождись статуса **Сопряжено** (Paired)

### Шаг 2: Узнай MAC-адрес

**Вариант A — через Параметры Windows:**

1. Открой **Параметры** → **Bluetooth и устройства**
2. Нажми на **Timebox-Evo** → **Свойства устройства** (или три точки → Свойства)
3. Найди поле с MAC-адресом (формат `XX:XX:XX:XX:XX:XX`)

**Вариант B — через командную строку:**

```powershell
Get-PnpDevice -Class Bluetooth | Where-Object { $_.FriendlyName -like "*Timebox*" } | Select-Object FriendlyName, DeviceID
```

MAC-адрес будет в поле DeviceID (последние 12 символов, разделённые двоеточиями).

**Вариант C — через приложение Divoom:**

Если на телефоне стоит приложение Divoom, MAC-адрес виден в настройках подключённого устройства.

### Шаг 3: Укажи MAC-адрес в коде

Открой `draw_pixel.py` и задай свой MAC-адрес в переменной `MAC_ADDRESS`.

## Скрипты

### `draw_pixel.py` — один пиксель

```powershell
# Красный пиксель в центре (8, 8)
python draw_pixel.py

# Зелёный пиксель в левом верхнем углу
python draw_pixel.py 0 0 0 255 0

# Синий пиксель в координатах (5, 10)
python draw_pixel.py 5 10 0 0 255
```

### `draw.py` — произвольное изображение

Отредактируй сетку `GRID` прямо в файле (16×16 символов) и запусти:

```powershell
python draw.py
```

Доступные символы: `.` чёрный, `R` красный, `G` зелёный, `B` синий, `W` белый, `Y` жёлтый, `C` голубой, `M` пурпурный, `O` оранжевый.

### `voice.py` — голосовое управление цветом

Скажи слово-активатор **«коробка»** или **«колонка»**, затем название цвета (красный, зелёный, синий, белый, жёлтый…).

```powershell
python voice.py
```

### `voice_timer.py` — голосовой таймер

Скажи **«колонка, таймер на пять минут»** или **«таймер тридцать секунд»**. На экране колонки отображается обратный отсчёт, по завершении играет звуковой сигнал (`timer.mp3`). Для остановки скажи **«стоп»**.

```powershell
python voice_timer.py
```

#### Автозапуск при входе в Windows

В папке автозагрузки (`shell:startup`) лежит `voice_timer_autostart.vbs` — запускает `voice_timer.py` через `pythonw.exe` без консольного окна.

Проверить, что процесс запущен:

```powershell
Get-Process -Name pythonw -ErrorAction SilentlyContinue
```

### `listen.py` — мониторинг кнопок

Выводит в консоль сырые сообщения от колонки при нажатии кнопок. Удобно для отладки.

```powershell
python listen.py
```

### `move.py` — управление пикселем с клавиатуры

Перемещай белый пиксель по экрану клавишами `W`/`A`/`S`/`D`, `Q` — выход.

```powershell
python move.py
```

## Структура проекта

| `timer.mp3`                 | Звук окончания таймера                          |
| `vosk-model-small-ru-0.22/` | Модель распознавания русской речи (Vosk)        |

## Устранение проблем

| Проблема                    | Решение                                                             |
| --------------------------- | ------------------------------------------------------------------- |
| `OSError: [WinError 10051]` | Bluetooth выключен или адаптер отсутствует                          |
| `OSError: [WinError 10061]` | Устройство не сопряжено или выключено                               |
| `ConnectionRefusedError`    | Попробуй RFCOMM канал 2 (измени `RFCOMM_CHANNEL` в `draw_pixel.py`) |
| `Model not found`           | Распакуй модель Vosk рядом со скриптами                             |
| Пиксель не появляется       | Убедись, что колонка не в режиме будильника/таймера                 |

## Протокол

Экран Timebox Evo — матрица 16×16 пикселей. Координаты: `(0,0)` — левый верхний, `(15,15)` — правый нижний. Связь по Bluetooth RFCOMM (Serial Port Profile).

Подробная документация: [PROTOCOL.md](https://github.com/RomRider/node-divoom-timebox-evo/blob/master/PROTOCOL.md)
