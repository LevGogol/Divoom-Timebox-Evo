# Divoom Timebox Evo

Управление Divoom Timebox Evo через Bluetooth: отрисовка изображений, голосовое управление яркостью и таймером с визуальным обратным отсчётом.

## Требования

- Python 3.11+
- Windows 10/11 с Bluetooth-адаптером
- Divoom Timebox Evo

```powershell
pip install -e .
pip install vosk pyaudio playsound==1.2.2
```

Модель распознавания речи (~45 МБ), скачать один раз:

```
https://alphacephei.com/vosk/models/vosk-model-small-ru-0.22.zip
```

Распаковать так, чтобы рядом со скриптами находилась папка `vosk-model-small-ru-0.22/`.

## Подключение колонки

### Шаг 1: Сопряжение

1. Включи Timebox Evo
2. Открой **Параметры Windows** → **Bluetooth и устройства** → **Добавить устройство**
3. Выбери **Bluetooth**
4. Найди **Timebox-Evo** в списке и нажми **Подключить**
5. Дождись статуса **Сопряжено** (Paired)

### Шаг 2: Узнай MAC-адрес

**Вариант A — через Параметры Windows:**

1. Открой **Параметры** → **Bluetooth и устройства**
2. Нажми на **Timebox-Evo** → **Свойства устройства** (или три точки → Свойства)
3. Найди поле с MAC-адресом (формат `XX:XX:XX:XX:XX:XX`)

**Вариант B — через командную строку:**

```powershell
Get-PnpDevice -Class Bluetooth | Where-Object { $_.FriendlyName -like "*Timebox*" } | Select-Object FriendlyName, DeviceID
```

MAC-адрес будет в поле DeviceID (последние 12 символов, разделённые двоеточиями).

**Вариант C — через приложение Divoom:**

Если на телефоне стоит приложение Divoom, MAC-адрес виден в настройках подключённого устройства.

### Шаг 3: Укажи MAC-адрес в коде

Открой `app/draw_pixel.py` и задай свой MAC-адрес в переменной `MAC_ADDRESS`.

## Запуск

После установки (`pip install -e .`) доступна команда `timebox`:

```powershell
timebox
```

## Голосовые команды

Запускается единственной командой `timebox`. Постоянно слушает микрофон и реагирует на команды со словом-активатором **«коробка»** или **«колонка»**.

При запуске яркость устанавливается на 100%. При выходе (Ctrl+C) и при переходе компьютера в спящий режим яркость сбрасывается в 0, при пробуждении — восстанавливается.

### Активация

Скажи **«коробка»** или **«колонка»** — на экране появится смайлик-подтверждение.

### Яркость

**«коробка, яркость пятьдесят»** — установить яркость в 50 (0–100). Принимаются числа словами («двадцать пять», «сто») и цифрами.

### Таймер

**«таймер на пять минут»**, **«таймер тридцать секунд»** — запустить обратный отсчёт. На экране отображается прогресс-бар (зелёный → жёлтый → красный). По истечении играет звук (`app/timer.mp3`) и экран гаснет.

### Стоп

**«стоп»** — остановить активный таймер. Экран переключается в режим часов.

> При естественном истечении таймера экран также переключается в режим часов.

## Структура проекта

```
run.py                          — точка входа, сборка зависимостей
app/
  app.py                        — App: основной цикл
  protocol.py                   — кодирование сообщений (изображение, яркость, таймер, часы…)
  draw_pixel.py                 — MAC_ADDRESS, RFCOMM_CHANNEL
  power.py                      — управление яркостью + хуки suspend/resume
  power_monitor.py              — WindowsPowerMonitor: WM_POWERBROADCAST через Win32
  timer.mp3                     — звук окончания таймера
  commander/
    voice_commander.py          — распознавание речи (Vosk) → очередь команд
  executor/
    console_executor.py         — вывод команд в консоль
    timebox_executor/
      timebox_executor.py       — разбор фраз, диспетчеризация команд
      comand/
        brightness_command.py   — установка яркости
        timer_command.py        — обратный отсчёт на экране
        stop_command.py         — остановка таймера + переключение в часы
        wakeup_command.py       — смайлик-подтверждение активации
        ignore_command.py       — нераспознанная команда
vosk-model-small-ru-0.22/       — модель распознавания русской речи (Vosk)
```

## Устранение проблем

| Проблема                    | Решение                                                                 |
| --------------------------- | ----------------------------------------------------------------------- |
| `OSError: [WinError 10051]` | Bluetooth выключен или адаптер отсутствует                              |
| `OSError: [WinError 10061]` | Устройство не сопряжено или выключено                                   |
| `ConnectionRefusedError`    | Попробуй RFCOMM канал 2 (измени `RFCOMM_CHANNEL` в `app/draw_pixel.py`) |
| `Model not found`           | Распакуй модель Vosk рядом со скриптами                                 |
| Пиксель не появляется       | Убедись, что колонка не в режиме будильника/таймера                     |

## Протокол

Экран Timebox Evo — матрица 16×16 пикселей. Координаты: `(0,0)` — левый верхний, `(15,15)` — правый нижний. Связь по Bluetooth RFCOMM (Serial Port Profile).

Подробная документация: [PROTOCOL.md](https://github.com/RomRider/node-divoom-timebox-evo/blob/master/PROTOCOL.md)
